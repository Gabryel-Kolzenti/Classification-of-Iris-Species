# Classificação de Espécies de Flores Iris

## Objetivo

O objetivo deste projeto é desenvolver um modelo de Machine Learning capaz de classificar espécies de flores Iris com base em características físicas das pétalas e sépalas.

As espécies previstas serão:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Tipo do Problema

Este é um problema de classificação supervisionada multiclasse, pois:
- possuímos exemplos rotulados;
- existem múltiplas categorias possíveis;
- queremos prever uma classe específica.

## Objetivo Técnico

Além da construção do modelo, este projeto busca demonstrar:
- análise exploratória;
- preparação de dados;
- escolha de métricas;
- validação do modelo;
- ajuste de hiperparâmetros;
- serialização do modelo para deploy.
